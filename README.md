# programmer-photography

This repository contains a VitePress site with:

- Home page
- Guide section
- Blog section

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the local dev server:

   ```bash
   npm run docs:dev
   ```

3. Build for production:

   ```bash
   npm run docs:build
   ```

4. Preview production build:

   ```bash
   npm run docs:preview
   ```
